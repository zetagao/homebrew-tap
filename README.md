# zetagao/homebrew-tap

Homebrew tap for [zetagao](https://github.com/zetagao)'s tools.

## Install

```bash
brew install --no-quarantine zetagao/tap/limithud
```

| Cask | Description |
| --- | --- |
| [`limithud`](https://github.com/zetagao/LimitHUD) | Menu-bar HUD for live Claude & Codex usage quotas |

The cask is updated automatically by [LimitHUD](https://github.com/zetagao/LimitHUD)'s release workflow.
